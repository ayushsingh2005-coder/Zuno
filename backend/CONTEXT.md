## Architecture Decisions:
- OTP в†’ Redis (setex 300s signup, 900s reset)
- Token Blacklist в†’ Redis (setex 86400s)
- Songs Cache в†’ Redis (1 hour)
- Profile pic в†’ Cloudinary (zuno/profiles)
- Songs в†’ Cloudinary (zuno/songs)
- Thumbnails в†’ Cloudinary (zuno/thumbnails)
- No blacklistToken model (Redis handle karta hai)
- Upstash returns parsed object (JSON.parse mat karo)
- Upstash returns number (String() se compare karo)
- mongoose { new: true } deprecated в†’ { returnDocument: "after" }
- User model name "User" capital U

## API Endpoints:

### Auth (Public):
POST /api/auth/send-otp
POST /api/auth/verify-otp
POST /api/auth/register
POST /api/auth/resend-otp
POST /api/auth/login
POST /api/auth/send-reset-otp
POST /api/auth/reset-password

### Auth (Protected):
GET  /api/auth/profile
POST /api/auth/logout

### User (Protected):
GET /api/user/profile
PUT /api/user/profile
PUT /api/user/profile/picture
PUT /api/user/change-password

### Songs (Public):
GET /api/songs
GET /api/songs/search?q=...
GET /api/songs/genre/:genre
GET /api/songs/:id

### Songs (Protected):
POST /api/songs/:id/play

### Songs (Admin):

### Albums (Public):

### Albums (Admin):
